* Intro
This file contains my NixOS system configuration in a literate format.
The focus of this NixOS config is light coding, audio production and a bit of gaming.

** To update the system
1. Makes changes in this file (not the one in etc/nixos/)
2. Tangle this file using org-bable-tangle() function
3. Run the update_nix_config.sh in the same directory to update the real nixos config

Every blocks needs to point at the configuration.nix for proper tangling, because of the {} and the start/end of the file.

** Managing previous generations (builds) of NixOS
*** List all generations
nix-env --list-generations
*** To delete all previous generations
sudo nix-collect-garbage -d

* Nix release version
This value determines the NixOS release from which the default settings for stateful data, like file locations and database versions on your system were taken. 
It‘s perfectly fine and recommended to leave this value at the release version of the first install of this system.
Before changing this value read the documentation for this option (e.g. man configuration.nix or on https://nixos.org/nixos/options.html).
#+BEGIN_SRC nix :tangle configuration.nix
      system.stateVersion = "25.05"; # Did you read the comment?
